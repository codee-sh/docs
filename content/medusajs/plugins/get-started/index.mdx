# Plugins

Plugins extend Medusa.js functionality with additional features and integrations.

## Available Plugins

- [Notifications](/medusajs/plugins/notifications) - Transactional notifications with templates for email and Slack.
- [Automations](/medusajs/plugins/automations) - Rule-based automations with triggers and actions for workflows.
