# Starters

Starters are pre-configured Medusa.js projects that provide a ready-to-use foundation for building your e-commerce store. Each starter includes a specific set of plugins, configurations, and customizations tailored for a particular use case.

## How it works

Each starter is a complete Medusa.js project that you can clone and run locally. Starters are reviewed by the Codee team and community to ensure quality and compatibility.

### Review Status

- **Reviewed** — The starter has been reviewed and approved by the Codee team. It meets quality standards and is compatible with the specified Medusa.js version.
- **In Review** — The starter has been submitted and is currently being reviewed. It may not yet meet all quality standards.

## Submitting a Starter

Want to share your starter with the community? Open a pull request to the [codee-docs](https://github.com/codee-sh/codee-docs) repository with your starter documentation following the template below.

### Required Information

| Field | Description |
|-------|-------------|
| **Status** | `Reviewed` or `In Review` |
| **Repository URL** | Link to the public Git repository |
| **Description** | Full description including installation and usage instructions |
| **Version** | Current version of the starter |
| **Author** | Company name or author's full name |
